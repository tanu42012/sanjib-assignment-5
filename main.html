Question-1: What are the different ways to select an element in the DOM?

1. By ID

const element = document.getElementById("myId");

2. By ClassName

const elements = document.getElementsByClassName("myClass");

3. By Tag Name

const elements = document.getElementsByTagName("div");

4. By CSS Selector (Single Element)

const element = document.querySelector(".myClass");

5. By CSS Selector (Multiple Elements)

const elements = document.querySelectorAll(".myClass");

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.InnerHtml

To get HTML content inside an element.

2.innerText

To get the visible text inside an element.

3.textContent

To get all text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where instead of adding event listeners to multiple child elements, you attach a single event listener to a parent element and handle events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a mechanism in the DOM where an event triggered on a child element propagates (bubbles up) to its parent elements until it reaches the root

Question-5: How do you create, add, and remove elements using JavaScript?

1. Creating Elements

To create a new element, use the document.createElement() method.

const newElement = document.createElement("div");
newElement.textContent = "Hello, I am a new div!";

2. Adding Elements to the DOM

Once an element is created, you can insert it into the DOM using various methods.
appendChild()

document.body.appendChild(newElement);

3. Removing Elements

1.remove()
newElement.remove();

2.sremoveChild()
const parent = document.getElementById("parent");
const child = document.getElementById("child");
parent.removeChild(child);
